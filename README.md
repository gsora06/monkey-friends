## Setup

To run locally:
1. Clone this repository
2. Open `index.html` in your browser
3. Or run a local server: `python -m http.server 8000`

## API Keys

For real monkey photos, you'll need free API keys:
- [Unsplash API](https://unsplash.com/developers) - 50 requests/hour free
- [Pixabay API](https://pixabay.com/api/docs/) - Unlimited free

Update the `CONFIG` object in the HTML file with your keys.
